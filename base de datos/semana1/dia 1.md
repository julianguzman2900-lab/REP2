# Base de datos 

SQL = Structured Query Languaje o Lenguaje de consulta estructurada 

SQL es un lenguaje de consultas 

## Conceptos fundamnetales 
- Entidad: representa un objeto del mundo real que tiene relevancia en el sistema(`cliente`)
- Tabla: forma en la que se almacena la informacion de  una entidad en la base de datos 
- Filas (Registros) : Corresponde a una instancia concreta de la identidad (`cliente especifico`)
- Columnas (atributos) : Define una caracteristica de la entidad (`nombre, direccion,precio`)
Llave(clave) primaria(PK): Identificador **unico** para cada fila en una tabla
- Llave(clave) foranea(FK): Relacion entre tablas, enlazando con la lleve primaria de otra tabla


DBMS: DataBase Managment System
